
# UDP-Calculator-Chat

A simple real-time chat and calculator application using the **UDP protocol**. This project demonstrates how clients can communicate over a network using UDP for sending messages and performing basic calculations.

---

## Features

- Real-time chat between multiple clients using UDP.
- Basic calculator functionality integrated into the chat.
- Lightweight network communication without using threads.
- Easy-to-understand Python implementation for learning purposes.
  
---

## Usage

- Run the server:
  ```bash
  python server.py
  ```

- Run the client(s):
  ```bash
  python client.py
  ```

Start chatting and performing calculations in real-time.
